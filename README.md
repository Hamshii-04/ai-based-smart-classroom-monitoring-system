# AI-Based Smart Classroom Monitoring System


## Overview


AI-Based Smart Classroom Monitoring System is a computer vision-powered application designed to automate classroom attendance, monitor student engagement, and provide real-time classroom analytics. The system uses face recognition and AI-based monitoring techniques to improve classroom management and reduce manual effort.

## Features

* Automatic attendance tracking using face recognition
  
* Real-time student identification

* Student attention and engagement monitoring

* Admin dashboard for attendance and analytics

* Secure student data management

* Real-time classroom monitoring

* Attendance reports and insights

## Installation

### Clone Repository

```bash
git clone https://github.com/Hamshii-04/ai-based-smart-classroom-monitoring-system.git
cd ai-based-smart-classroom-monitoring-system
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python start_system.py
```
